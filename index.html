Projects

TOPQuote

I was given an existing template and asked to rework it to match the needs of TOPQuote, a SaaS offering for the fire, alarm, suppression, and sprinkler industry. This was my first project getting really familiar with Bootstrap and jQuery to make the design both responsive and interactive, and I enjoyed manipulating the CSS to give the whole site a sleek, clean, professional feel.

Skills: HTML/CSS, Bootstrap, JavaScript, jQuery, Flexbox, Responsive Web Design

Tools: Visual Studio Pro, TFS / Azure DevOps

Super Sticky Notes

This was my first truly functional app made with React, and I’m so pleased with how it turned out! Users are able to add, edit, and delete notes, which are searchable using the input field in the header. Notes are saved to local storage so that you can return to them later, even if you close your browser.

Skills: HTML/CSS, React & JSX

Tools: CodeSandbox

Unplugged Retreat

This fully responsive website for the Unplugged Retreat uses Flexbox and mobile-first design principles to make a clean, simple, and engaging website that looks beautiful on any mobile, tablet, and desktop device.

Skills: HTML/CSS, Git, Flexbox, Responsive Web Design

Tools: Atom, GitHub

Guess the Word Game

This “Guess the Word” game uses JavaScript and a random word API to generate a mystery word and allow users a limited number of letter guesses to see if they can…yep…guess the word! JavaScript keeps the game functional by limiting repeated guesses, tracking the remaining number of guesses, and notifying the user when they’ve won or lost. Give it a spin!

Skills: HTML/CSS, JavaScript, Git

Tools: APIs, Atom, GitHub

GitHub Repo Gallery

This gallery of my GitHub repos was made with JavaScript. API calls gather the repo info from GitHub, and some additional JavaScript code allows the user to search for repos by name as well as click on repos to see more info about each one.

Skills: HTML/CSS, JavaScript, Git

Tools: APIs, Atom, GitHub
